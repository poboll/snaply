## 🌟 核心特性

- **🎨 Y2K 复古 UI** - 像素级还原 Windows 98/2000 经典界面
- **📤 智能上传** - 拖拽上传、批量处理、实时进度
- **🗄️ 多存储后端** - 本地 / S3 / MinIO 灵活切换
- **🤖 AI 自动打标** - 支持 5 个 AI 提供商，智能识别图片内容
- **🏷️ 标签管理** - 手动 + AI 自动打标，快速分类
- **⚙️ 高级配置** - WebP 转换、CDN、EXIF 保留
